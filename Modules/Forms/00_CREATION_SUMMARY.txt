DATABASE SCHEMA
================================================================================

Tables Created:
  1. forms
     - UUID id (primary key)
     - Guild ID, Name, Description
     - Questions (JSONB array)
     - Response channel, Max responses
     - Settings (one_per_user, dm_confirm, is_active)
     - Timestamps

  2. form_responses
     - UUID id (primary key)
     - Form ID (foreign key)
     - User ID, Answers (JSONB)
     - Status (pending/approved/denied)
     - Review metadata
     - Timestamps

  3. forms_config
     - Guild ID (primary key)
     - Module settings
     - Notification channel
     - Timestamps

INTEGRATION INSTRUCTIONS
================================================================================

1. Run Database Schema:
   psql -U postgres -d your_db -f DATABASE_SCHEMA.sql

2. Add Environment Variables:
   FORM_BASE_URL=https://your-domain.com
   API_BASE_URL=http://localhost:3000

3. Initialize in Bot:
   import formsModule from './Modules/Forms';
   formsModule.initialize(client, dbPool, eventBus);

4. Mount Express Routes:
   import formsRouter from './Modules/Forms/web/routes';
   app.use('/forms', formsRouter);

5. Set User Identification:
   Middleware to set x-user-id header (JWT, OAuth, session, etc)

See QUICK_START.md for 5-minute setup
See INTEGRATION_GUIDE.md for detailed instructions

USAGE EXAMPLES
================================================================================

Create a form:
  /formcreate name:Job Application description:Join our team! responsechannel:#applications oneperuser:true dmconfirm:true

Add questions:
  /formedit formid:<id> addquestion label:Full Name type:short_text required:true
  /formedit formid:<id> addquestion label:Email type:email required:true
  /formedit formid:<id> addquestion label:Position type:dropdown required:true
  /formedit formid:<id> addquestion label:Tell us about yourself type:long_text required:false

Get form link:
  /form form:Job Application

Review responses:
  /formreview formid:<id> status:pending
