================================================================================
CONFIGURATION SCHEMA
================================================================================

InviteConfig (stored as JSONB in guild_settings):
{
  "enabled": boolean,
  "trackJoins": boolean,
  "trackLeaves": boolean,
  "trackFakes": boolean,
  "fakeAccountAgeDays": number (1-365),
  "fakeLeaveHours": number (1-720),
  "logChannelId": string (optional),
  "announceJoins": boolean,
  "announceChannelId": string (optional)
}

Defaults:
{
  "enabled": true,
  "trackJoins": true,
  "trackLeaves": true,
  "trackFakes": true,
  "fakeAccountAgeDays": 7,
  "fakeLeaveHours": 24,
  "announceJoins": false
}
