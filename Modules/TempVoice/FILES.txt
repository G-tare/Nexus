================================================================================
                          FILE MODIFICATION GUIDE
================================================================================

To customize the module:

1. Change channel name template
   File: helpers.ts, line 14
   DEFAULT_CONFIG.nameTemplate

2. Change default bitrate
   File: helpers.ts, line 21
   DEFAULT_CONFIG.bitrate

3. Change cooldown
   File: helpers.ts, line 12
   DEFAULT_CONFIG.cooldownSeconds

4. Change delete after empty time
   File: helpers.ts, line 17
   DEFAULT_CONFIG.deleteAfterEmpty

5. Change cleanup interval
   File: cleanup.ts, line 19
   CHECK_INTERVAL constant

6. Add new user permission checks
   File: Any core command, validation section

7. Change embed colors
   File: Any command, EmbedBuilder color

8. Add audit logging integration
   File: helpers.ts, auditLog function
