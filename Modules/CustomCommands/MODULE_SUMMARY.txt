================================================================================
                         DATABASE SCHEMA
================================================================================

Table: custom_commands
Columns: 15
- id (UUID) - Primary key
- guild_id (VARCHAR) - Foreign key
- name (VARCHAR) - Command name
- aliases (JSONB) - Alternative names
- response (TEXT) - Response template
- embed_response (BOOLEAN) - Use embed format
- required_role_id (VARCHAR) - Role requirement
- cooldown (INTEGER) - Seconds
- use_count (INTEGER) - Usage counter
- created_by (VARCHAR) - Creator user ID
- created_at (TIMESTAMP) - Creation time
- dm (BOOLEAN) - Send as DM
- ephemeral (BOOLEAN) - Hide from others
- delete_invocation (BOOLEAN) - Delete trigger
- add_reaction (VARCHAR) - Reaction emoji
- allowed_channels (JSONB) - Channel whitelist

Table: custom_commands_config
Columns: 6
- guild_id (VARCHAR) - Primary key
- enabled (BOOLEAN) - Module enabled
- prefix (VARCHAR) - Command prefix
- max_commands (INTEGER) - Command limit
- allow_slash (BOOLEAN) - Slash registration
- created_at (TIMESTAMP)
- updated_at (TIMESTAMP)

Table: command_cooldowns
Columns: 4
- guild_id (VARCHAR) - Composite key
- command_id (UUID) - Composite key
- user_id (VARCHAR) - Composite key
- cooldown_expires_at (TIMESTAMP) - Expiration

Indexes:
- idx_custom_commands_guild_id
- idx_custom_commands_name
- idx_cooldowns_expires

================================================================================
                      TROUBLESHOOTING GUIDE
================================================================================

Issue: Command not triggering
Solution:
- Check /cconfig view (enabled status)
- Verify correct prefix
- Check for cooldown message
- Verify command name spelling

Issue: Variables not replacing
Solution:
- Check variable syntax (no spaces)
- Use /cvariables to verify availability
- Ensure context exists (e.g., channel topic may be null)
- Check for special characters

Issue: Permission denied
Solution:
- Verify ManageGuild permission
- Check permission path configuration
- Verify user roles

Issue: Database errors
Solution:
- Check PostgreSQL connection
- Verify tables created
- Check connection pool status
- Review error logs

Issue: High latency
Solution:
- Check database connection
- Reduce response text length
- Avoid complex nested variables
- Monitor event listener frequency
